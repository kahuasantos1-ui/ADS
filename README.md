# 🎓 ADS — Site Institucional do Curso

Site institucional sobre o curso de **Análise e Desenvolvimento de Sistemas** (ADS), feito como projeto da faculdade.

🌐 **Demo:** [sitefaculdade.vercel.app](https://sitefaculdade.vercel.app)

## 🎯 O que é

Página informativa sobre o curso de ADS, apresentando:

- Visão geral do curso
- Disciplinas e grade curricular
- Mercado de trabalho
- Identidade visual focada em estudantes

## 🛠️ Stack

- **HTML5**
- **CSS3**
- **Vercel** — deploy contínuo

## 🚀 Como rodar localmente

```bash
git clone https://github.com/kahuasantos1-ui/ADS.git
cd ADS
# abrir index.html no navegador
```
